# 🎙️ Text-to-Speech

A simple Text-to-Speech application built using Python, gTTS, and Gradio.

## 🚀 Features

- Convert text into speech
- Download generated audio
- Simple and interactive web interface
- Fast speech generation
- Supports English language

## 🛠️ Technologies Used

- Python
- gTTS (Google Text-to-Speech)
- Gradio

## 📂 Project Structure

```
Text-to-Speech/
│── app.py
│── Text_to_Speech.ipynb
│── requirements.txt
│── README.md
└── .gitignore
```

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run

```bash
python app.py
```

## 📌 Use Cases

- Accessibility
- Language learning
- Reading text aloud
- Educational applications
